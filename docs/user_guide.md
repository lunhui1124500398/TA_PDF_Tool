# TA PDF Tool 使用说明

## 1. 现在可以做什么

当前这版已经能直接用于手动批改 PDF 作业，支持：

1. 从作业目录创建批改会话。
2. 在左侧队列里切换学生，查看已完成和未完成数量。
3. 单页查看 PDF，支持翻页和缩放。
4. 快速放置 `✓`、`△`、`✗`。
5. 添加中文文本批注和分数批注。
6. 选中批注后修改内容、颜色、字号、粗细、位置、宽高。
7. 直接拖动批注，支持更符合直觉的上下移动。
8. 批语库复用，最近使用会优先展示。
9. 导出当前学生或全部学生的批改后 PDF。
10. 页面内自定义全部快捷键，删除默认支持 `Delete` 和 `Backspace`。
11. 默认工具、颜色、字号、会话输入框和缩放会保存在本机浏览器里，下次打开会继续沿用。

## 2. 如何启动

推荐直接运行：

```powershell
cd D:\Zhujiao\ta_pdf_tool
.\start_mvp.ps1
```

如果想手动启动：

```powershell
cd D:\Zhujiao\ta_pdf_tool
& 'D:\Conda_Data\envs\ta_ocr_gpu\python.exe' -m uvicorn app:app --reload --host 127.0.0.1 --port 8000
```

启动后，在浏览器打开：

```text
http://127.0.0.1:8000
```

## 3. 第一次使用

进入页面后：

1. 在顶部“作业目录”里填 `D:\Zhujiao\HW-4&5`，或点击“选择文件夹”直接用系统目录选择框挑选。
2. 会话名可以保持默认 `hw45-mvp`。
3. 点击“创建 / 重建会话”。

创建成功后，系统会扫描目录中的 PDF，并自动生成学生队列。

## 4. 如何开始批改

### 4.1 切换学生

- 点击左侧队列中的学生。
- 使用顶部“上一份 / 下一份”按钮。
- 使用你自己配置的快捷键。

### 4.2 翻页和缩放

- 使用顶部“上一页 / 下一页 / 缩小 / 放大”按钮。
- 或使用你配置的翻页快捷键。

### 4.3 放置符号批注

1. 在右侧工具区选择 `✓`、`△` 或 `✗`。
2. 点击 PDF 页面上的目标位置。

只有 `✓`、`△`、`✗` 会按符号批注处理。只要内容不是这三种符号，系统都会自动按文本批注保存和导出。

### 4.4 放置文本批注

1. 可以点击“文本”工具，也可以直接按你配置的“切换到文本”快捷键，或在 PDF 空白处右键。
2. 点击 PDF 页面后，会直接在页面内进入文字输入，不需要再去浏览器输入框里填。
3. 输入批语后按 `Enter` 保存；如果这次是通过快捷键或右键临时切到文本模式，保存后会自动回到之前的批改工具。
4. 如果需要手动换行，可以按 `Shift+Enter`。

### 4.5 放置分数

1. 选择“分数”工具后，页面会先自动回到第 1 页，方便统一在首页录入总分。
2. 点击 PDF 页面。
3. 输入例如 `98`、`8/10`、`-2` 这样的内容，然后按 `Enter` 保存。

## 5. 如何修改批注样式和位置

### 5.1 直接拖动

- 用鼠标拖动页面上的批注即可。
- 左右拖动对应左右移动，上下拖动对应上下移动。
- 双击文本类批注，可以直接在页面上进入文字编辑模式，类似 Photoshop 的文本编辑逻辑。
- 如果批注当前处于选中状态，点击页面空白处只会取消选中，不会额外再放一个新批注。
- 按 `Esc` 也可以快速取消当前选中或退出页面内文字编辑。

### 5.2 右侧精调

点击某条批注后，可在右侧“选中批注”中修改：

- 内容
- 颜色
- 字号
- 粗细
- `X`
- `Y`
- 宽
- 高

其中 `Y` 按照页面从上往下的习惯来理解，和拖动方向一致。

## 6. 批语复用

右侧有两块：

1. 最近批语
2. 常用批语

使用方式：

1. 如果当前已经选中一个文本批注，点击某条批语会直接覆盖进去。
2. 如果当前没有选中文本批注，点击某条批语会先放入“文字草稿”，接着你再去页面上点击落点即可。

## 7. 快捷键

右侧“快捷键”面板支持页面内修改全部快捷键。

使用方式：

1. 点击某个动作右侧的按键按钮。
2. 直接按下新的按键组合。
3. 页面会立即保存到本地浏览器存储。

除了快捷键之外，右侧默认样式、当前工具、作业目录、会话名和缩放也会一起保存在本机浏览器里。

其中“切换到文本”默认更适合做临时文字批注：按下后点击空白处开始输入，按 `Enter` 保存后会自动回到你刚才的批改工具。

默认包括：

- 上一份 / 下一份
- 上一页 / 下一页
- 切换到 `✓` / `△` / `✗`
- 切换到文本 / 分数
- 删除批注
- 标记完成 / 进行中
- 导出当前 / 导出全部

删除批注默认支持：

- `Delete`
- `Backspace`

## 8. 如何导出

### 导出当前学生

点击“导出当前”。

### 导出全部学生

点击“导出全部”。

导出结果会保存到：

```text
D:\Zhujiao\HW-4&5\graded
```

总成绩表会导出为：

```text
D:\Zhujiao\HW-4&5\graded\scores.csv
```

## 9. 本地数据保存位置

批改过程中的本地数据保存在：

```text
D:\Zhujiao\ta_pdf_tool\data
```

主要包括：

- `session.json`：当前会话与队列进度
- `annotations.json`：所有批注
- `comment_library.md`：常用批语库
- `comment_usage.json`：最近使用记录

## 10. 当前范围

这版是可直接开工的手动批改 MVP，不依赖 OCR。

当前主流程聚焦在：

1. 看 PDF
2. 手动落批注
3. 调整样式和位置
4. 复用批语
5. 导出结果

`legacy/ocr_extract.py`、`legacy/segment_questions.py`、`legacy/doubao_grader.py` 是之前 OCR / AI 原型链路留下来的脚本，不是当前这版开始使用所必需的部分。
