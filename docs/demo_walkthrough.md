# 5 分钟 Demo 演示稿

这份稿子适合你给其他助教现场演示，目标是让他们在 5 分钟内明白这套工具怎么用。

## Demo 前准备

1. 准备一个只包含 1-2 份 PDF 的演示文件夹。
2. 启动程序：

```powershell
cd D:\Zhujiao\ta_pdf_tool
.\start_mvp.ps1
```

3. 浏览器打开 `http://127.0.0.1:8000`

## 演示顺序

### 1. 创建会话

话术：

“这里不是手敲路径也行，可以直接点‘选择文件夹’。选好作业文件夹后，点‘创建 / 重建会话’，左边就会出现这次要批的学生队列。”

要展示的动作：

- 点击“选择文件夹”
- 选一个演示目录
- 点击“创建 / 重建会话”

### 2. 切换学生和翻页

话术：

“左边是队列，上面能看到当前学生、当前页、完成数量。可以点按钮，也可以用快捷键上一份、下一份、上一页、下一页。”

要展示的动作：

- 点“上一份 / 下一份”
- 点“上一页 / 下一页”

### 3. 快速打勾叉

话术：

“最常见的是打勾、半对、打叉。选中工具后，直接点页面就落批注。”

要展示的动作：

- 切到 `✓`
- 在 PDF 上点击两处
- 切到 `△` 和 `✗` 各演示一次

### 4. 快速加文字批注

话术：

“最顺手的方式不是点右侧输入框，而是直接在 PDF 空白处右键，马上就在页面里开始打字。按 `Enter` 保存后，会自动回到刚才的批改工具，可以继续点勾叉。”

要展示的动作：

- 先选 `✓`
- 在空白处右键
- 输入一句中文批语
- 按 `Enter`
- 再点击页面落一个 `✓`

### 5. 调整批注样式和位置

话术：

“批注可以直接拖动，也可以在右侧细调颜色、字号、粗细和位置。双击文本批注可以直接改内容。”

要展示的动作：

- 拖动一条批注
- 在右侧改颜色和字号
- 双击一条文本批注，改字，再按 `Enter`

### 6. 录入分数并导出

话术：

“选分数工具时会自动回到第一页，方便统一在首页记总分。最后可以导出当前学生，也可以一键导出全部。”

要展示的动作：

- 切到“分数”
- 观察页面自动回到第 1 页
- 输入 `98`
- 点击“导出当前”

## 结尾总结

建议最后只强调这 4 句话：

1. 先建会话
2. 勾叉直接点
3. 右键就是文字批注
4. 分数在第一页录，最后一键导出
